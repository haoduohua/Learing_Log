# Learing_Log
Make your own Learning Log, and list the topics you're learning about.

# Requirements
- python==3.6
- Django==1.8.4
- django-bootstrap3==6.2.2

# Useage
- 下载该仓库
- 按照需要依赖模块，最好安装在虚拟环境下
- 第一次使用需要迁移模型
- 在仓目录下执行 `python manage.py runserver`

# Note
- Code comes from "《Python编程：从入门到实践》 作者：Eric Matthes"
- 暂不支持Heroku部署
